# 🎵 Spotify Clone

![Amazon Clone Banner](https://raw.githubusercontent.com/ronik-bajakke/SpotifyClone/main/banner.png)

A Spotify Web Player Clone built using HTML5, CSS3, and Font Awesome.
This project replicates the layout and functionality of Spotify’s web interface — including sidebar navigation, library, playlist cards, and an elegant music player.

---

## 🌐 Live Demo  

👉 **[Visit the Live Site](https://github.com/Soham-patil18/SpotifyClone.git)**  

---

## 🎯 Project Overview 
 
This Spotify Clone is a front-end UI recreation that demonstrates skills in responsive web design, modern CSS, and interface layout structuring.
It captures Spotify’s signature look with custom playlists, navigation, and player controls — built entirely using static technologies.

---

## 🧠 Features  

- 🎧 Fully responsive design  
- 🧭 Sidebar with Home, Search, and Library
- 💿 Dynamic music cards and trending sections 
- 📀 Custom-built bottom music player with progress and volume bars
- 🎚️ Smooth playback controls (play, pause, forward, backward)  
- 💻 Built using only HTML & CSS (no frameworks)

---

## 🏗️ Built With  

- **HTML5** — for structuring the webpage  
- **CSS3** — for styling, flexbox layout, and responsive design  
- **Font Awesome** — for icons used in navigation and footer  
- **Google Fonts** — Montserrat typography

## 🚀 How to Run Locally  

1. Clone the repository  
   ```bash
   git clone https://github.com/Soham-patil18/SpotifyClone.git
   ```  
2. Open the project folder  
   ```bash
   cd SpotifyClone
   ```  
3. Open `index.html` in any web browser
